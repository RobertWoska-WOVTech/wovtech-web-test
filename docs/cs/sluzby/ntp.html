NTP: časová infrastruktura

Přesný a konzistentní čas je základem bezpečnosti, auditovatelnosti a řešení incidentů. NTP patří mezi tier-0 služby, jejichž selhání má řetězový dopad.

Co typicky řešíme

  • návrh zdrojů času (redundance, důvěryhodnost, on-prem / cloud / hybrid)
  • monitoring synchronizace a časového driftu
  • hardening NTP a bezpečné konfigurace klientů
  • postupy při incidentu souvisejícím s nesprávným časem
